To install the open-mediakit transcoder, you will need the following : 

- a Linux or *Nix install with a webserver and php5 (version 5.3 minimum) and php libraries (php5-curl)

- ffmpeg and imagemagick WITH proprietary codecs if you want to be able to encode to mp4, mp3 etc.

To do this, under debian Linux, use debian-multimedia repository as follow 
(add this line in /etc/apt/sources.list.d/debian-multimedia.list )

deb http://debian.octopuce.fr/debian-multimedia squeeze-backports main contrib non-free
deb http://debian.octopuce.fr/debian squeeze-backports main contrib non-free

then launch, as root : 

# aptitude update
# aptitude install -t squeeze-backports ffmpeg libavcodec-extra-53 libavdevice-extra-53 libavfilter-extra-2 libavformat-extra-53 libavutil-extra-51

(later, when we will support document parsing, we may need pdfutils, htmltotext, unoconv ...)
